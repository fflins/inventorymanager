# Gerenciador de Produtos Genéricos

Este é um projeto de um gerenciador de produtos genéricos para realizar operações CRUD com um sistema de Autenticaçao e Roles.

## Tecnologias Utilizadas

- **Back-end**:
    - Spring Boot
    - Spring Security
    - JPA/Hibernate
    - MySQL
    - JWT (JSON Web Tokens)

- **Front-end**:
    - Angular

## Pré-requisitos

- [Java 17 ou superior]
- [MySQL]
- [Node.js]
- [Angular CLI]
